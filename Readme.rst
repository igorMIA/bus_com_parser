Deploying
--------------

Using docker-compose
^^^^^^^^^^^^^^^^^^^^^

Before first project start, you should create your env file, or rename .env.example to .env

*  to start project run::

    $ docker-compose up django

*  to run tests run::

    $ docker-compose up tests
